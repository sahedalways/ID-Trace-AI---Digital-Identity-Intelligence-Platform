# ID Trace AI

ID Trace AI is a Vanilla PHP based OSINT (Open Source Intelligence) platform that helps analyze an individual's public digital footprint across multiple online sources and generates intelligent identity reports.

The platform is designed for identity verification, online research, due diligence, recruitment, fraud prevention, and digital trust analysis.

---

## Features

- Facebook Profile Lookup
- Public Profile Detection
- Username Intelligence
- Digital Footprint Analysis
- Cross Platform Identity Search
- AI Generated Intelligence Reports
- Report Credits System
- Secure Payment Integration
- User Dashboard
- Report History
- Subscription & Billing
- Responsive UI
- Email Notifications

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/idtrace-ai.git
```

2. Move the project to your web server.

3. Import the SQL database.

4. Configure database credentials.

```php
DB_HOST=
DB_NAME=
DB_USER=
DB_PASS=
```

5. Start Apache & MySQL.

6. Open the project in your browser.
